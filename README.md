# Ella Musaieva - Portfolio

Creative Technologist portfolio featuring interactive 3D models, research documentation, and project showcase.

## Live Site

Visit: [uttertosh.com](https://uttertosh.com)

## Features

- **Interactive 3D Model** - WebGL/Three.js model with glass materials
- **AI News Ticker** - Rotating tech news feed
- **Research Documentation** - Photogrammetry and 3D scanning projects
- **Responsive Design** - Works on all devices

## Quick Start

### GitHub Pages Deployment

1. Fork or clone this repository
2. Go to Settings → Pages
3. Source: Deploy from branch
4. Branch: `main` (root folder)
5. Save

Your site will be live at: `https://yourusername.github.io/repository-name/`

### Local Development

```bash
# Serve with Python
python -m http.server 8000

# Or with Node.js
npx serve

# Visit http://localhost:8000
```
